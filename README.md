# BAD — Basic Analysis of Data

BAD (Basic Analysis of Data) is a lightweight full-stack web application that enables users to upload datasets, automatically clean the data, analyze key statistics, and generate professional PDF reports with charts — all in a single click.

The application is designed for fast exploratory data analysis without requiring any setup, authentication, or prior data science knowledge.

## Features

- Upload CSV and Excel files
- Automatic data cleaning
  - Removes duplicates and empty rows
  - Trims invalid values
- Column-level statistical analysis
- Auto-generated PDF reports with visual charts
- Download cleaned Excel output
- Clean and responsive UI
- No authentication required
